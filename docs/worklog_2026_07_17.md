### 6. RBAC Auth Skeleton
- **`src/lib/prisma.ts`** — Prisma client singleton
- **`src/lib/jwt.ts`** — Dual JWT: `jose` (Edge Runtime) + `jsonwebtoken` (Node Runtime), HS256, 24h expiry
- **`src/lib/rbac.ts`** — `hasPermission()` with wildcard support (`*` matches any submodule/page)
- **`src/middleware.ts`** — Edge middleware: JWT verification via jose, injects user context headers
- **`src/app/api/auth/login/route.ts`** — POST login, seeds demo admin role + permission, issues JWT
- **`src/app/api/protected/test/route.ts`** — GET protected route, checks `system.test.demo.view` permission
